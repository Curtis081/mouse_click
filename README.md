## Introduction(簡介)
The plot_wafer_map function is designed to visualize wafer maps, providing a clear representation of the status of individual dies on a semiconductor wafer. It creates a graphical plot where each die is represented by a square, and their status is color-coded for easy interpretation.

plot_wafer_map 函數旨在視覺化晶圓圖，清晰呈現半導體晶圓上各晶元的狀態。它創建一個圖形，其中每個晶元由一個正方形表示，其狀態以顏色編碼，方便解讀。
